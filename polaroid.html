polaroid.js

Transform any picture into a polaroid photo. It's free, easy and fast!

polaroid.js works with every image size but for best results use images with web resolution, i.e. up to 1024 pixels wide.
2. Download your polaroid image

How it works

No server side scripting. Everything done on client side. Thanks to polaroid.js and HTML5 technologies such as drag & drop and file API.
That's right, there are no privacy issues as there is no uploading to our server - the images you use do not leave your local computer, that's why this is so fast.
